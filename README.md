# Change_Detection_NDWI
วิเคราะห์พื้นที่ตลิ่งริมน้ำโขง การกัดเซาะเเละทับถม ด้วยการใช้ NDWI เเละติดตามการเปลี่ยนเเปลงราย 10 ปี เเละภาพรวมตั้งเเต่ปี ค.ศ. 1993 ถึง 2026
พื้นที่ศึกษาใช้ ROI_Polygon สี่เหลี่ยม บริเวณสะพานข้ามเเม่น้ำโขงไทย - ลาว 
ทำการวิเคราะห์ NDWI เพื่อเเยกพื้นดินเเห้งเเล้ง พื้นที่ชุ่มชื้น เเละเเหล่งน้ำ
ติดตามการเปลี่ยนเเปลงด้วยการใช้สูตร บวกลบ ค่า NDWI 9yh'gg9j -0.4 57' 0.4 ในพื้นที่ศึกษา โดยการเเบ่งประเภทการกัดเซาะเเละทับถมออกเป็น 7 ประเภท
